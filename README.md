🚀 Space Mission Analysis Dashboard

This is a Power BI dashboard project where I analyzed historical space mission data and created interactive visualizations to 
understand mission trends, company performance, launch locations, and success rates.

#Dashboard Overview
Total Missions: 4.555K
Success Rate: 89.93%
Companies: 62
Rockets: 370

#Visualizations Used
Mission Count by Company
Success Rate by Year
Global Launch Locations Map
KPI Cards
Year Slicer
Tools Used
Power BI
Power Query
DAX
CSV Dataset

#Key Insights
RVSN USSR conducted the highest number of missions.
Overall mission success rate is close to 90%.
Mission success has improved significantly over time.
Launch activities are concentrated in a few major regions across the world.
